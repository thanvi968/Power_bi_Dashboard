# Data Jobs Dashboard w/ Power Bi

![Dashboard Page 1](../Images/Project1_Page1)
## Introduction

This dashboard was created for **Job seekers, Job Transitions, and Job Swappers** to solve common porblem: information about the data job market is scattered and hard to grasp. Using a *real-world dataset of 2024 data science job postings*(including titles, salaries, and locatrions), this project provides a single, easy-to-use interface to explore market trends and compensations.

## skills Showcased

- **Data Transformations (ETL) with Power Query:** Cleaned, shaped, and prepared the raw data for analysis by handling blanks, changing data types, and creating new columns.
- **Implicit Measures:** Formulated measures to derive jey insights and KPIs like `Median Yearly Salary` and `Job Count`.
- **Core Charts:** Utilized **Column, Bar, Line,** and **Area CHrts** to compare job counts and track trends over time.
- **KPI Indicators & Tbles:** Used **Cards** to display key metrics and **Tables** to provide granular,sortable data.
- **Geospatial Analysis:** Leverged *8Map Charts** to visualize the global distribution of jobs.
- **Dashborad Design:** Designed an intuitive and visually appealing layout, exploring both common and uncommon chart types to best tell data story.
-**Interactive Reporting** To dynamically filter the report by Job Title.
    - **Slicers:** To dynamically filter the report by Job Title.
    - **Button & Bookmarks:** To create a seamless navigation ecperience.
    - **Drill-Through** To navigate from a hugh-level summary to a contexual, detailed view.

This dashboard showcases how Power Bi can transform raw job posting data into a powerful tool for career analysis.It allows users to slice, filter, and drill through data to make informed decisions about their career paths.
